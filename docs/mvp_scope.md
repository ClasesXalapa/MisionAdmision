# Alcance del MVP

## Implementado hasta v0.6.0

- Recordatorio Web Push opcional mediante Firebase Cloud Messaging.
- Consentimiento, prueba y desactivación desde la PWA.

- Flutter Web y GitHub Pages.
- Banco global de preguntas.
- Examen libre.
- Reto diario programado y automático.
- Reanudación local durante el mismo día.
- Racha, récord, escudos y rangos.
- Cards de recursos y filtros.
- Seguimiento local de recursos vistos y completados.
- Sincronización remota desde `content/index.json`.
- Descarga selectiva por versión.
- Última versión válida local del contenido.
- Fallback automático a assets incluidos.
- Actualización manual y estados sin conexión.
- Protección de retos e intentos pendientes ante bancos incompatibles.
- Service worker propio y caché PWA controlada.
- Experiencia guiada de instalación.
- Detección y activación explícita de actualizaciones.
- Validadores y pruebas automatizadas.

## Pendiente dentro del MVP

- Notificación diaria mediante Firebase Cloud Messaging.
- Flujo de consentimiento para notificaciones.
- Mecanismo seguro de envío programado.
- Pruebas reales en navegadores y dispositivos objetivo.

## Fuera del MVP

- Cuentas de usuario.
- Sincronización entre dispositivos.
- Panel administrativo completo.
- Rankings globales.
- Pagos.
- Comentarios o red social.
- Analíticas individuales de alumnos.
