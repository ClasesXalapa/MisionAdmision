- ruta `/help`;
- acceso desde la barra superior y la pantalla de inicio;
- versión de aplicación visible;
- detección aproximada de navegador y sistema;
- estado de conexión, HTTPS, PWA y service worker;
- estimación de almacenamiento usado y disponible;
- versiones activas de contenido;
- estado general de Firebase y permisos de notificaciones;
- resumen local sin respuestas del alumno;
- reporte copiable;
- descarga de diagnóstico JSON;
- instrucciones para reportar errores;
- puente JavaScript aislado y pruebas Node.
